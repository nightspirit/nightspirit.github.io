Arduino + Xbee + 2x16 Chars LCD

02-19-2010

After we made the first prototype we found the critical shortcoming of Bluetooth and we thought it is not a good idea to apply BT on our garments.

The main reason is because BT communication has to pair two devices before working. Once we pair two of them it is not easy interrupting with third garment. We consider BT is not an ideal protocol for peer communication.

So we want to try something else. We aim our solution to Xbee. Xbee and Arduino are perfect for each others because there are existing modules on the market such as Xbee shield, Xbee breakout for easy implementing. Simply plug it to the Arduino and you are ready to go.

If you are not going to mess around with Zbee 2.5 protocol then everything is easy. The AT command is just a text-based command line interface so all you have to do is follow the instruction and send out the serial message.

To start the AT command mode you have to wait 1000 millisecond for security gap (Just delay(1000);) and immediately send the serial string “+++” to Xbee. Afterward, if you get “OK” response then you’ve successfully entered the AT command mode.
