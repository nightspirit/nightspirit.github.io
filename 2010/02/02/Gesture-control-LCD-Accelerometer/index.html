Gesture control menu (LCD+Accelerometer)

02-02-2010

For second garment, we decided to put second display on the sleeve. Because, we consider it is not convenient for user to look at his/her belly all the time. A simple text-based indicator would be nice for user to track his/her dragon stats.

However, the LCD display took out 6 pins that make the system has not enough pins for other devices. We must discard some devices to keep the LCD display going. On the second thoughts, if we only use it as an indicator that seems too wasted its power. So why don’t we combine the accelerometer and display to make a gesture control menu instead of original fabric buttons.

With the inspiration, I started to develop the program for it. And here is what I’ve done so far.

Basically, the left and right change the options and the up and down decide go back or enter next level. Through the accelerometer tracking I can tell the swing direction of the user’s wrist. Afterwards I can assign corresponding actions to each event.

This feature can make user intuitively select the function they want. At the mean time it can replace the fabric buttons and save us enough pins for using other components.
